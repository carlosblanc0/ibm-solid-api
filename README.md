## Run Locally

Clone the project

```bash
  git clone  git@github.com:carlosblanc0/ibm-solid-api.git
```

Go to the project directory

```bash
  cd ibm-solid-api
```

Install dependencies

```bash
npm install
```

Spin up a Mailtrap account free tier for demoing purposes

```bash
touch .env
```

```bash
# MongoDB Atlas
USER:<>
PASS:<>

# SMTP port
PORT=25
```

Start the server

```bash
  npm start
```
